KYC是什麼？為什麼決定成單質量

AXA視角

  • KYC = 合規信息收集＋需求澄清＋風險適配
  • 每次面談產出可追溯的紀要與同意（記錄-復述-確認）
  • 方案價值 = 客戶目標 × 現金流可行 × 風險承受

常見失誤

  • 先講產品，後了解現金流/負債/用款時間
  • 未確認優先級與時間表（里程碑/到期點）
  • 無紀要，無法複盤與跟進

原則：重述客戶原話，關鍵數字必記下。

活動：最近一次失單，缺了哪塊信息？

分層標準（生命週期 × 資產級別）

生命週期（7段）

  • 1 單身起步｜0–5年工齡
  • 2 新婚/備孕｜1–3年
  • 3 幼兒期｜0–6歲
  • 4 學齡期｜6–18歲
  • 5 夾心期｜供樓/養小/養老並行
  • 6 空巢/臨退｜50–60歲
  • 7 退休/傳承｜60+

資產級別（A–D）

  • A 基礎：可動用淨資產 < HKD 1M；可投現金流率 < 10%
  • B 進階：HKD 1–5M；可投 10–20%
  • C 資產：HKD 5–20M；可投 15–30%
  • D 高淨值：> HKD 20M；可投 > 20%

可動用淨資產＝現金＋有價投資－短期負債

補充標籤

  • 現金流波動：月收入標準差/均值 ≥ 0.5 → 高波動
  • 負債率：本息/稅後收入 > 45% → 優先現金流防禦
  • 流動性：備用金 6–12個月（自僱取 12–18個月）
定位規則：先週期→再資產級→加標籤（波動/負債/稅務/法人）

維度 問題 標籤/門檻

🪄 互動練習1：幫你的鄰座「分層定位」

目標：用2分鐘問3個問題，把對方放進矩陣格子裡

步驟（4分鐘）

  1. 兩人一組，A先問B，B再問A。
  2. 每人問對方3個問題（見右欄）。
  3. 根據答案，把對方定位到「生命週期 × 資產級別」矩陣（如 2B、3A）。
  4. 互相核對：「你把我分到__，我覺得合理嗎？」

提示：不用精準，重點是「敢問、敢判、敢說」。

3個必問題（任選3問）

  • 你現在處於哪個人生階段？
    （例：單身／新婚／有幼兒／供樓中）
  • 你每月可自由支配的錢，大概能存多少？
    （例：＜3K → A級｜3–8K → B級）
  • 你有沒有房貸或大額負債？
    （本息＞收入45% → 加「高負債」標籤）
  • 你覺得自己收入穩定嗎？
    （波動大 → 加「高波動」標籤）

舉例：
「你28歲、月存5K、無房貸 → 1B」
「你32歲、有幼兒、月存4K、房貸佔40% → 3B + 高負債」

教學提示：分錯沒關係，重點是熟悉「提問→分類→標籤」流程

需求分析｜場景卡（1–2）

場景1｜單身上班族（剛工作2–3年）

  • 目標常見：先有基本醫療和小額存錢，別影響生活

必問3句

  1. 你到手工資大概多少？最淡的一個月大概多少？
  2. 每月固定花費（房租/交通/吃飯）大概多少？
  3. 現在手上有多少現金是隨時能用的？

口算1步：能存的錢 ≈ 工資 − 固定花費（看是否≥3000–5000）

結論：先把醫療做起來，再留3–6個月生活費，剩下每月少量做短期儲蓄。

話術：先把看病和急用錢安排好，再慢慢加儲蓄，壓力不會大。

場景2｜新婚或準備要小孩

  • 目標常見：擔心收入有變化，怕懷孕生產費用高

必問3句

  1. 家裡主要是誰賺錢？如果有變化，生活會受多大影響？
  2. 每月房租/房貸和父母支持大概多少？
  3. 產檢、分娩、坐月子希望用什麼標準？

口算1步：家庭「不可少的錢」 = 房租/房貸 + 基本生活 + 父母支持

結論：先給賺錢的那位安排身故和重病保障，產前把醫療升級，儲蓄先小額開始。

話術：先保住家庭「不可少的錢」，孩子來了也不慌。

新人提示：每頁兩張卡，問3句、算1步、收1句。

需求分析｜場景卡（3–4）

場景3｜有幼兒（0–6歲）

  • 目標常見：托育/學前花費上升，開始想教育金

必問3句

  1. 每月必要花費裡，孩子相關大概佔多少？
  2. 如果你或配偶停工3個月，哪筆開支最擔心斷掉？
  3. 教育金準備打算幾年後用？一次用還是每年用？

口算1步：教育金起步額 = 計劃每年學費 × 2（先做一半，壓力較小）

結論：先做雙人的身故/重病保障，教育金先分期小額開始，等收入穩定再加碼。

話術：先把最怕斷掉的開支兜住，教育金我們慢慢攢，別一下子太重。

場景4｜學齡期（6–18歲），考慮海外

  • 目標常見：3–5年內要一筆學費，怕匯率和投資波動

必問3句

  1. 目標學校在哪個國家？首筆學費和生活費大概多少？
  2. 家裡每月能穩定拿出多少錢做教育金？
  3. 如果投資遇到下跌，你希望學費資金還能不能動？

口算1步：首筆準備金 = 1年學費＋1年生活費（盡量用美元/目標幣種）

結論：把教育金單獨放一桶，用相對穩的方式鎖定時間和幣種，醫療保障同時升級。

話術：學費是「必須要用的錢」，先把第一年鎖住，後面每年再接力。

教學提醒：先鎖「必需用的錢」，再談收益與彈性。

需求分析｜場景卡（5–6）

場景5｜「夾心期」（供房、養小、顧父母）

  • 目標常見：房貸壓力大，怕失業或患重病

必問3句

  1. 每月房貸本息多少？如果沒收入幾個月，房子會不會出問題？
  2. 家裡誰是主要收入來源？另一個人能不能頂一部分？
  3. 現在的醫療和重病保障各有多少？

口算1步：房貸安全邊際 = 手上現金 ÷ 房貸月供（至少要撐6個月為佳）

結論：優先用「隨貸款逐年減少」的保障把房貸兜住，再補醫療/重病，儲蓄小額起步。

話術：先保證「房子不出事」，其他再逐步加，不要拖家裡後腿。

場景6｜臨近退休（50–60歲）

  • 目標常見：鎖定退休現金流，擔心醫療費用

必問3句

  1. 每月基本生活費多少？退休後希望保留到什麼水平？
  2. 現有投資裡，有沒有一部分希望變成「每月固定領」？
  3. 有沒有擔心需要長期照護的情況（比如父母那代）？

口算1步：退休「底線」 = 基本生活費 × 12 × 10年（先保證前10年）

結論：把部分資金換成按月/按年領取的方式，醫療和大病續保要關注續保穩定。

話術：先鎖住每月能領的錢，退休心裡更有底。

收尾話術：您的處境是「場景X」。我做3個預算檔供選。週四或週五，哪天15分鐘方便過一眼？

需求分析前：3個動作讓客戶願意說

先建立安全感，客戶才會給你真實數字

✅ 動作1：開場「減壓聲明」

「今天不是推產品，是幫您整理家庭財務健康檢查表，任何問題您都可以跳過或之後補。」

為什麼有效？
→ 客戶怕被推銷，你先解除武裝，他才敢說真話。

✅ 動作2：先聊「非數字目標」

「如果未來5年，您最想為家人完成的一件事是什麼？」
（例：孩子留學、父母醫療無憂、提早退休）

→ 從情感目標切入，比直接問收入更容易打開話匣子。

✅ 動作3：用「假設情境」問數字

  • 不問：「你月入多少？」→ 太直接
  • 改問：「如果未來12個月收入減少20%，哪些開支是您絕對不想砍的？」
  • 或問：「萬一突發住院，您希望自付額控制在多少內？」

→ 用「情境」包裝「數字」，客戶不覺得被審查，反而覺得你在幫他模擬風險。

記住：客戶不抗拒「規劃」，抗拒「被審問」。

行動：面談前深呼吸，默念「我是顧問，不是銷售」

需求分析實戰：客戶卡點 × 精準應對

目標：不被問題卡住，把抗拒轉為需求確認

客戶常見5大卡點

  • 「我沒時間細談」 → 時間壓力
  • 「收入不穩定，不敢規劃」 → 波動恐懼
  • 「已有保險，不用再買」 → 認知盲區
  • 「錢要留着供樓/養老」 → 優先級錯配
  • 「等我穩定一點再說」 → 拖延心理

核心原則：
不反駁、不教育、不催促 —— 把問題轉成「共同解決方案」

一對一破解話術

客戶說 你回應
「我沒時間」 「我們只聚焦1個最想保障的目標，10分鐘搞定，其他下次補。」
「收入不穩」 「正因不穩，才要設計彈性方案——收入高時多存，低時可暫停，不影響保障。」
「已有保險」 「太好了！我們快速核對三件事：
1. 保額是否覆蓋缺口？
2. 是否涵蓋最新家庭責任？
3. 現金流是否仍可負擔？」
「錢要留着供樓」 「理解！我們優先設計『供樓保障型』方案——萬一出事，保險金自動幫還房貸，不動您現金。」
「等穩定再說」 「完全尊重！但保障有『健康門檻』和『年齡溢價』，現在鎖定基礎，未來升級更省力。」
行動：把5句回應存入手機備忘錄，面談遇阻立即調用

🪄 互動練習2：用「假設情境」問出真實數字

目標：不問「你月入多少」，而是問「如果…你會怎麼辦？」

步驟（5分鐘）

  1. 兩人一組，A扮演客戶，B扮演顧問。
  2. B從下方選1個假設情境問A（不要直接問數字！）
  3. A誠實回答（可虛構），B必須記錄兩個數字：
    → 金額 M｜→ 時間點 D
  4. 交換角色，再練一次。

舉例：
問：「如果未來12個月收入減少20%，你最不想砍掉哪筆開支？」
答：「孩子的學費，每月HKD 3000。」
→ 記錄：M=3000, D=每月

6個安全假設情境（任選）

  • 如果突發住院，你希望自付醫療費不超過多少？
  • 如果收入中斷3個月，你存款能撐多久？
  • 如果孩子3年後要留學，你希望那時手上有多少準備金？
  • 如果房貸利率上升1%，你每月多付多少仍可接受？
  • 如果要為父母準備醫療備用金，你希望金額是多少？
  • 如果只能保障一個家庭成員，你會先保誰？預算多少？

關鍵：問完後，一定要復述數字：
「我幫你記下：學費每月HKD 3000，對嗎？」

行動：把「假設情境」抄在筆記本第一頁，面談前溫習

三、設計方案｜新人三步法（填空就能用）

不用懂投資組合，記住：先保底 → 再對齊目標 → 最後談增值

📌 第一步：保底檔（必做）

口訣：
「急用錢＋看病錢＋斷收錢」

  • 急用錢 = 6個月生活費（放活期/貨基）
  • 看病錢 = 醫療險 + 重疾險（保額 ≥ 1年收入）
  • 斷收錢 = 定期壽險（保額 = 負債 + 5年家庭支出）

推薦產品：愛唯守危疾保障、智尊守慧醫療、真智惠保

📌 第二步：目標檔（主推）

口訣：
「鎖時間 + 鎖金額 + 鎖幣種」

  • 鎖時間：幾年後要用？→ 選繳費期/提取期匹配
  • 鎖金額：目標多少？→ 每月/每年存多少能達標？
  • 鎖幣種：港元/美元？→ 選對應幣種保單

例：孩子6年後留學，每年要40萬 → 選5年繳儲蓄險，第6年起分年提取

推薦產品：智活系列、盈家系列、真智珍寳

📌 第三步：進階檔（可選）

口訣：
「加受益人 + 加信託 + 加貸款備用」

  • 加受益人：指定誰、什麼條件領錢（例：滿18歲、完成大學）
  • 加信託：大額保單用「保險金信託」防揮霍/爭產
  • 加貸款：保單現金價值可借80%，應急不賣單

新人提示：進階檔是加分題，客戶有興趣再提。

推薦產品：摯匯儲畜計劃、尊尚盈家II

行動：把「三步口訣」抄在筆記本首頁，見客前默念三遍
